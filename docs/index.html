Features

豊富な機能で
トレーニングを完全サポート

カレンダー管理
月別カレンダーでトレーニング履歴を一目で確認。 完了・進行中・未記録の日を色分けして表示するので、 継続状況がひと目でわかります。
トレーニング記録
種目・セット数・重量・回数を手軽に記録。 胸・背中・肩・腕・脚の5部位から種目を選び、 複数種目をまとめて登録できます。
筋肉疲労マップ
過去7日間のトレーニング量を「半減期モデル（48時間/50%回復）」で計算。 人体の正面・背面 SVG 図に疲労度をカラーで表示します。 オーバートレーニングの防止に役立ちます。
トレーニングタイマー
全体経過時間とインターバルタイマーを同時管理。 インターバル終了時はビープ音 + バイブレーションでお知らせ。 インターバル時間は±30/60秒で調整可能。
トレーニング詳細レポート
1日のトレーニング詳細を一覧表示。 総ボリューム（kg）・推定消費カロリー・種目コース・ 所要時間を自動算出してレポートします。
マルチログイン対応
メール＋パスワードのほか、Google / LINE の ソーシャルログインに対応。 初回ログイン時に自動アカウント作成し、 次回からワンタップでログインできます。
パスワードリセット
パスワードを忘れた場合はメールアドレスで再設定。 有効期限30分のワンタイムリンクを送信し、 安全にパスワードを更新できます。
管理者機能
ジムオーナー・管理者は全会員のトレーニングデータを閲覧可能。 ユーザーの有効/無効・権限管理もできます。 部位別ボリュームのグラフ表示にも対応。
ログイン維持 (Remember-me)
「ログインを保持する」チェックボックスで 7日間ログイン状態を維持。 毎回ログインする手間を省きます。
How It Works

科学的な
疲労度管理

過去7日間のトレーニングデータをもとに、部位ごとの疲労度を計算。 疲労が50%回復するまでの時間（半減期）を 48時間として算出するため、 休息の必要な筋肉がひと目でわかります。

  • 人体 SVG で視覚的に確認
  • 疲労度 0〜100% をカラーで表示
  • 部位別ボリューム・セット数を一覧
  • オーバートレーニング防止に活用

Security

ジム導入を見据えた
堅牢なセキュリティ

会員の個人情報・トレーニングデータを守るため、 12 項目のセキュリティ対策を実装しています。

CSRF 保護
全フォーム送信・API リクエストに CSRF トークンを要求。第三者サイトからの不正操作を防止します。
パスワードハッシュ (BCrypt)
パスワードは BCrypt で安全にハッシュ化。大文字・小文字・数字・記号の混在を必須としています。
ブルートフォース対策
ログイン失敗5回で15分間アカウントをロック。辞書攻撃やパスワード総当たりを自動でブロックします。
セッション管理
ログイン成功時にセッション ID を再生成（固定化攻撃対策）。同一ユーザーのセッションを1つに制限します。
IDOR 対策（所有者チェック）
全データ操作で「自分のデータか」を検証。他ユーザーのトレーニングデータを参照・変更・削除できません。
セキュリティヘッダー
HSTS・CSP・X-Frame-Options・Permissions-Policy など主要なセキュリティヘッダーを網羅的に設定。
Cookie セキュリティ
セッション Cookie に HttpOnly・Secure・SameSite=Lax を設定。本番環境では HTTPS のみで送信されます。
エラー情報の秘匿化
スタックトレース・例外メッセージをレスポンスに含めない設定。攻撃者への情報漏洩を防ぎます。
パストラバーサル対策
CSV バッチ取り込み時にファイルパスを正規化・検証。許可ディレクトリ外へのアクセスを遮断します。
メールヘッダーインジェクション対策
パスワードリセットメール送信時に CR/LF を除去。メールヘッダーの改ざんによるスパム悪用を防止します。
データ保護期間 7年
退会後も7年間はデータを物理削除しない保護期間を設定。誤削除・不正削除からのデータ復元に対応。
設定値の暗号化 (Jasypt)
DB パスワードなど機密性の高い設定値を Jasypt で暗号化して管理。ソースコードへのハードコードを禁止。

ジム導入のメリット

スタッフ・会員双方にとってメリットのある機能を多数搭載。

会員のモチベーション維持
カレンダーで継続を可視化。トレーニング履歴が一目でわかるので、 会員の継続率向上・退会防止に貢献します。
全会員データの一元管理
管理者ダッシュボードから全会員のトレーニング状況を確認。 部位別ボリュームのグラフで進捗を把握できます。

退会・アカウント停止・権限変更をブラウザから操作可能。 スタッフ向け管理者アカウントの発行もできます。
